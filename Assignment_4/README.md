# Assignment 4 – TSP Optimization (SA and GA)

## Overview
This project solves the Traveling Salesman Problem (TSP) using:
- Simulated Annealing (SA)
- Genetic Algorithm (GA)

## Requirements
Python 3.x

Required libraries:
- math
- random
- matplotlib
- dataclasses
- typing

## How to Run

### If using Jupyter:
Open the notebook:
Assignment_4.ipynb

Run all cells.

## Outputs
The program generates:
- Best SA tour and length
- Best GA tour and length
- SA ΔE histograms
- GA population average fitness plot
- SA vs GA comparison plots
